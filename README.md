# Influencer Marketplace Backend

This repository contains the **Backend API** for the **Influencer Marketplace** application. It is built using:

- ✅ **Spring Boot** — powerful Java backend framework
- ✅ **Spring Data JPA** — for ORM and database interaction
- ✅ **Spring Security** — for robust authentication and authorization

## 📌 Overview

This backend handles core business logic and data management for the Influencer Marketplace platform, enabling:

- User registration and login
- Role-based access control (e.g., Influencer, Brand, Admin)
- Influencer and brand profile management
- Campaign creation, management, and collaboration requests
- RESTful API endpoints consumed by the frontend
- Data persistence with relational database using JPA/Hibernate

## 🛠️ Tech Stack

- Java 8
- Spring Boot 2.5.6
- Spring Data JPA
- Spring Security
- Hibernate
- MySQL
- Lombok
- Maven
